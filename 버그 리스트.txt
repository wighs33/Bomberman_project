[심각한 버그] - 나중에 해결
1. 이름이 같은 플레이어가 있을 시 이동 불가능! -> 아이디 중복 사용 막아야 함
2. 정원이 모두 꽉 찼을때 다른 또 다른 플레이어 접속을 시도하면 서버, 클라이언트 모두 정지함 -> 정원 초과시 접속 막는 코드 필요
3. 게임 시작후 플레이어 접속 막기

[미미한 버그]
1. 여러명의 유저 같은 아이디 접속가능
2. 유저 한명 (또는 여러명이) 접속하고 아이디 입력 실패 후 나가게 되면 플레이어 인덱스 넘어감 (ex) 3p가 로그인 실패하면 다음 플레이어는 4p로 접속하게됨)
-> 즉, 서버에서 클라의 접속종료를 알 수 가 없어서 생기는 문제
3. 플레이어가 로그인 후 나가고 다른 플레이어가 접속을 하면 아이디가 '' 로 표시가 됨

[개선할 수 있는 사항]
1. 새로운 플레이어 접속시 DB에 저장하기
2. 비번 만들기

[해결사항]
1. 움직임 부드럽게 변경
2. 레디 만들기
